# AI Security Port Scanner
A lightweight web-based port scanner built with Python (Flask) and Nmap. 
Designed for security researchers to perform quick reconnaissance on local network assets.

## Features
- Scans common TCP ports (1-1024).
- Web interface for ease of use.
- Built-in Nmap integration.

## Usage
1. Install dependencies: `pip install flask python-nmap`
2. Run the scanner: `python app.py`
3. Access at `http://127.0.0.1:5000`
